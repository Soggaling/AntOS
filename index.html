getting.started
INSTALL GUIDE
01

Download the ISO

Grab the latest AntOS-v1.0.iso from the GitHub Releases page above. ~1.5GB download.

02

Flash to USB or load in VM

Use balenaEtcher or Rufus to write to a USB drive. Or load directly in VirtualBox / VMware.

03

Boot from ISO

Select "AntOS — Enter the Temple" from the GRUB menu. Auto-login will drop you into XFCE.

04

Enter the Temple

Open a terminal. See the credits. Type askgod. You are home.
